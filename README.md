# Ghost Intel Pro
Un outil multi-fonctions en Python pour l'OSINT et le tracking.

## Fonctionnalités
- Scanner de pseudonymes ultra-rapide (Asynchrone)
- Tracker d'adresses IP (Bientôt disponible)
- Analyse de numéros de téléphone (Bientôt disponible)

## Installation dans Termux
```bash
git clone https://github.com/housseini1960/ghost_intel.py.git
cd ghost_intel.py
pip install aiohttp
python ghost_intel.py
```
